[ what is this ]

Beam Steering,
explained simply.

Imagine pointing a flashlight beam in any direction — without moving your hand. That's phased array beam steering. Instead of physically rotating an antenna, we electronically shift the timing of each element so the signal points exactly where we want.

This tool lets you play with the parameters that control that. Move a slider. Watch the beam react.
